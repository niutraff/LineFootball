# LineFootball

## Тестирование приложения

Инструкция по тестированию приложения и управлению статусами модерации.

### Важно: VPN/Гео
Все запросы необходимо выполнять из указанной гео-локации с соответствующим VPN.
**Текущая гео:** IT (Германия)

### Требования

- Перейти на [ReqBin](https://reqbin.com/curl) — онлайн инструмент для выполнения cURL запросов
- Данные авторизации уже встроены в команды ниже

### Получение текущих настроек

Для получения текущего статуса модерации выполните GET запрос:

```bash
curl --request GET \
  --url http://174.129.199.77/config/line-football-stats/settings \
  --header 'Content-Type: application/json' \
  --header 'Authorization: Basic YWRtaW46a3JNOE80ODRXS21s'
```

**Шаги:**
1. Перейти на [ReqBin](https://reqbin.com/curl)
2. Вставить команду выше
3. Нажать "RUN"
4. Посмотреть текущие настройки и статусы модерации

### Изменение статуса модерации

Чтобы изменить статус модерации:

1. **Определить элемент**: В ответе GET запроса элементы обозначены номерами `"2"`, `"3"` и т.д.

2. **Изменить значение moderation**:
   - `"moderation": true` = На проверке (Review In Progress)
   - `"moderation": false` = Одобрено / Не на проверке

3. **Выполнить POST запрос** (пример для элементов "2" и "3"):

```bash
curl --request POST \
  --url http://174.129.199.77/config/line-football-stats/settings \
  --header 'Content-Type: application/json' \
  --header 'Authorization: Basic YWRtaW46a3JNOE80ODRXS21s' \
  --data '{
    "pdfUrl": "https://www.freeprivacypolicy.com/live/b7705891-8cec-4a3e-bb53-a001dd18d159",
    "companyId": "69981faf11aa29aec49dc42d",
    "2": { "moderation": true },
    "3": { "moderation": false }
  }'
```

**Шаги:**
1. Перейти на [ReqBin](https://reqbin.com/curl)
2. Вставить команду выше
3. **Изменить значения `"moderation"`** по необходимости:
   - Поменять `true` на `false` или наоборот в зависимости от нужного статуса
   - Можно изменить несколько элементов в одном запросе
4. Нажать "RUN"

### Рабочий процесс

1. **Проверить статус**: Выполнить GET запрос чтобы увидеть текущие статусы
2. **Обновить статус**: Выполнить POST запрос с изменёнными значениями moderation
3. **Проверить изменения**: Выполнить GET запрос снова для подтверждения
